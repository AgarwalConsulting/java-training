# Common

* Create a new Maven Project

```java
  mvn archetype:generate \
  -DgroupId=com.algogrit.java \
  -DartifactId=quickstart \
  -DarchetypeArtifactId=maven-archetype-quickstart \
  -DinteractiveMode=false \
  -DarchetypeVersion=1.4
```
